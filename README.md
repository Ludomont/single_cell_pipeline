# single_cell_pipeline
This repository contains the pipeline for different single cell analysis of CD8 T cells related to different experiments.
In particular contains:
- scRNA-seq pre-processing pipeline with Scanpy
- scATAC-seq pre-processing pipeline with SNAPATAC2
- multi-omic data integration with MOFA+
- scTCR-seq analysis and downstream analysis with Scirpy
- Automated data annotation with celltypist
- Metacells method
- HINT-ATAC pipeline
